BELIEVID

Description

BelieVid was built with the aim of being a One Stop Shop for anything related to Forensic Analysis and Verification of Digital Media. This project was one among the winners of Smart India Hackathon 2020.

Features

Web Application

  • Twitter Search: Verifies the authenticity of a tweet.
  • Image Metadata: Provides data describing size, color depth, the image resolution, geo location,  creation date, the shutter speed, and other data.
  • Reverse Image Search: Uses the image provided by the user as a search parameter to find similar images.
  • Photoshop Detection: Helps to figure out if the image has been photoshopped or not, and provides heatmap for the photoshopped areas.
  • Forensic Analysis: Performs various image analysis like JPEG Ghost, color space transformation, Min-max transformation, Error Level Analysis.
  • Screenshot OCR: Reads the text in the image to counter fake news and provide links to users from fact checking websites.
  • Video Metadata: Similar to image metadata.
  • Deepfake Detection: Deepfake Analysis provides deepfake probability with 22 frames showing the best face and the score for each.
  • Keyframe: Extracting the important frames from the video to perform further image analysis in order to gain more insights.
  • Chatbot: Integrated chatbot for users to understand the analysis performed, features available and for ease of use of BelieVId.

Browser Plugin

Mobile Application

The mobile application provides the same functionalities as the website application on the go.It also allows the sharing of URLs and images from messaging platforms, video streaming platforms and search engines to ease the users' experience in using the application. 

AI ChatBot

Using the messaging platform application Telegram, we have developed a chatbot through which all the questions regarding the image analysis can be understood by the user to help them come up with a conclusion. Links are also provided to the user to fact-check the claims in fake forwards. It is also integrated into our website as a widget.
